About NebulaX

An experimental OS with a quantum-event kernel

NebulaX is designed from the ground up with no reliance on existing operating systems. It features a spatial memory fabric and protocol-free device model to explore new ways of interacting with hardware.

Architecture Highlights

Key components currently under development:

  • Quantum Event Kernel
  • Spatial Memory Fabric
  • Protocol-Free Drivers
  • Starlight Language Runtime
  • Polychrome 3‑D GUI
